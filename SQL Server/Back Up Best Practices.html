Back Up Best Practices
  • Run DBCC regulary.
  • Verify CHECKSUM.
  • Run Verify Only.
  • Run a restore.

Factors That Improve Backup Compression Ratio

  • High free space in .mdf files → More compression.
  • Low fragmentation & well-structured data → Better compression.
  • Fewer LOB (Large Object) data types (TEXT, IMAGE, VARBINARY(MAX)) → LOB data is harder to compress.
  • Row/Page Compression enabled → Already compressed data may reduce backup compression benefits.
