Enable Instant File Initialization
  1. Go to Local Security Policy.

  2. Navigate to Local Policies > User Rights Agreement.

  3. In the Perform Volume Maintenance Tasks, add the SQL Server service account.

  4. Restart SQL Server service.

  5. Test.
    6. DBCC TRACEON (3004); -- Instant data file initialization info messages.
DBCC TRACEON (3605); -- Send DBCC output to the error log.

CREATE DATABASE [testDB];
  6. There should be no logs in the error log pertaining zeroing.
