Installing Windows Server 2016 Standard Edition (Desktop Experience)













Setting Up Basic Configuration

  1. On Server Manager:
    • Disable Firewall
    • Enable Remote Desktop
    • Set up Date & Time, if necessary.
  2. Rename Computer


  3. Reboot computer.

Configuring Domain Controller

  1. Configure IP address.


  2. On Server Manager:
    • Go to "All Server".
    • Right-click on the server name.
    • Choose "Add Role and Feature".
    • Choose "Role-based or feature-based installation".
    • Select the server.
    • Choose "Active Directory Domain Services" (choose Add Features).
    • Choose "DNS Server" (choose Add Features).
    • Finish the wizard.
    • Click "Promote this server to a domain controller".


  3. On the Active Directory Domain Services Configuration Wizard:
    • Deployment Configuration
      • Choose "Add New Forest".
      • Specify Root Domain name (ex: eapb.com).
    • Domain Controller Options
      • Configure password (for Administrator).
    • Finish the wizard. The computer will then reboot.
  4. Configure AD accounts.
    • Open Server Manager.
    • On tools, choose "Active Directory Users and Computers".
    • Create users/group/etc.

Joining a Server to a Domain

  1. Open Server Manager.
  2. On the Local Server, click the computer name.
  3. Click Change.
  4. Choose Member of..domain.
